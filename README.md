# ShauryaSetu - Bridge to Valor

## About

ShauryaSetu (शौर्य सेतु) is a free educational platform that bridges civilians with the history, values, and human stories of the Indian Army through respectful, curiosity-driven learning.

## Features

- **Educational Articles** - Well-researched content about army life, regiments, and history
- **AI Assistant** - Educational Q&A about army structure, careers, and training
- **Career Guidance** - Information about joining through NDA, CDS, Technical Entry, and Agniveer
- **Gallantry Awards** - Database of Param Vir Chakra, Maha Vir Chakra, and other awards
- **Wars & History** - Documentation of India's military history since independence
- **Movies & Context** - Real vs dramatized breakdown of army-inspired films
- **Support Information** - Verified NGOs supporting soldiers' families

## Important Notice

ShauryaSetu is an independent educational platform. All information is sourced from publicly available records. This platform is **not affiliated** with the Indian Army or Government of India.

## Tech Stack

- React 18
- CSS3 with custom properties
- Local storage for user data
- Responsive design

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm start

# Build for production
npm run build
```
